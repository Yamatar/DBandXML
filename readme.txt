Инструкция по запуску проекта:
	1. Восстановите базу данных с помощью файла job_db.sql
	2. Запустите файл main.bat и следуйте дальнейшим инструкциям

Использованные технологии и библиотеки:
	Чтение XML файла - XML DOM
	Запись XML файла - JAXB
	Логирование - log4j
	База данных - PostgreSQL
	Соединение с базой данных - JDBC
	Бэкап БД - Iperius Backup

Файл настроек приложения находится по пути src/resources/config.properties
